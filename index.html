HTML = estrutura • CSS = visual • JS = comportamento. Sobre Astronomia🪐 e outras Ciências✨

Como cada parte funciona? 👀

  • HTML é o esqueleto: títulos, parágrafos, imagens e links (as "peças").
  • CSS é a roupa: cores, tamanhos, fontes e espaçamentos (o "estilo").
  • JavaScript é o cérebro: cliques, buscas, filtros e contadores (as "ações").

Dica de design: escolha uma paleta com 2–3 cores e use bem o espaço em branco. Títulos curtos, imagens em boa resolução e textos objetivos deixam tudo mais bonito e legível.
